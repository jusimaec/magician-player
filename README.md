# 🎴 Magician Player 다운로드 페이지

도서출판 역리원의 **Magician Player** 공식 다운로드 페이지입니다.

🔗 **다운로드 페이지:** https://jusimaec.github.io/magician-player

---

## 소개

이 페이지는 Magician Player 설치 파일을 쉽게 다운로드할 수 있도록 만들어진 페이지입니다.  
Dropbox에 업로드된 파일 목록을 자동으로 불러와 한 번의 클릭으로 다운로드할 수 있습니다.

---

## 사용 방법

1. 위 다운로드 페이지 링크 접속
2. 원하는 파일의 **DOWNLOAD** 버튼 클릭
3. 파일 자동 다운로드 시작 ✅

> 💡 **크롬(Chrome) 브라우저 사용을 권장합니다.**
